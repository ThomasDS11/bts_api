Bienvenue sur l'API GSB

  • Recuperer les infos d'un visiteur en fonction de son nom.

    • Sans parametre : connexion.php

    Avec parametre : connexion.php?name=Villechalane

  • Recuperer la liste des departements avec au moins un medecin

    • Sans parametre : DepartementbyCP.php

    Avec parametre : DepartementbyCP.php?cp=75000

  • Recuperer les visiteurs en fonction d'un departement

    • Sans parametre : DepartementbyVIS.php

    Avec parametre : DepartementbyVIS.php?visCp=75000

  • Recuperer tous les medicaments si aucun parametre, recupere les infos d'un medicament avec parametre

    • Sans parametre : MedicamentbyDEPOTLEGAL.php

    Avec parametre : MedicamentbyDEPOTLEGAL.php?depotLegal=3MYC7

  • Recuperer les infos des praticiens si aucun param, avec param, recupere les infos du praticien concerne

    • Sans parametre : PraticienbyID.php

    Avec parametre : PraticienbyID.php?id=1

  • Cree une fiche en base de donnees avec les informations renseigner

    • Avec parametre : createRapport.php?visMat=a17&praNum=4&date=2021-03-20&bilan=RAS&motif=Rapport Annuel
